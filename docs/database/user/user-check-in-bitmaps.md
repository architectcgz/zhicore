# user_check_in_bitmaps - 用户签到位图表

## 表信息

| 属性 | 值 |
|------|-----|
| 表名 | user_check_in_bitmaps |
| 描述 | 用户月度签到位图表，用于快速查询某月签到情况 |
| 主键 | (UserId, Year, Month) 复合主键 |

| 字段名 | 类型 | 可空 | 默认值 | 说明 |

| Month | int | 否 | - | 月份 |
| Days | int | 否 | 0 | 签到位图（每一位代表一天） |

## 位图说明

`Days` 字段使用位运算存储签到状态：
- 第 1 位（最低位）代表 1 号
- 第 2 位代表 2 号
- 以此类推，最多支持 31 天

### 示例

```
Days = 0b00000000000000000000000000000111 (7)
表示 1号、2号、3号 已签到

Days = 0b00000000000000000000000010000001 (129)
表示 1号、8号 已签到
```

### 操作方法

```csharp
// 判断某天是否签到
bool IsCheckedIn(int day) => (Days & (1 << (day - 1))) != 0;

// 设置某天签到
void SetCheckIn(int day) => Days |= (1 << (day - 1));

// 取消某天签到
void ClearCheckIn(int day) => Days &= ~(1 << (day - 1));
```
